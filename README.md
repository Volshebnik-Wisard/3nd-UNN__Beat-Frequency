# Визуализация биений частот

Данный проект представляет собой MFC-приложение для визуализации интерференции (биений) двух гармонических сигналов. Реализованы два различных подхода к отображению графиков, распределенные по разным веткам репозитория. Вычисление сигнала:  

$$
\cos(\omega \cdot x + \varphi) + \cos((\omega + \Delta\omega) \cdot x)
$$

## Различия в реализациях

### Ветка Pixels: Выполняет попиксельное масштабирование через вычисление коэффициентов масштаба.
- Динамическое определение графика по фактическим значениям функции (`max_y`, `min_y`)
- Построчное рисование графика (`DrawLine` между точками)
- Дискретизация с фиксированным шагом (`step = 0.001`)
- Использование таймера (`WM_TIMER`) для анимации вместо отдельного потока

### Ветка Matrix: Использует матричные преобразования GDI+ для масштабирования и трансформации координатной сетки.
- Применение `Gdiplus::Matrix` для трансформации мировых координат в координаты окна
- Единое определение мирового окна (`worldRect = {0, -2, 2, 4}`)
- Автомасштабирование подписей по оси Y на основе логарифмического анализа данных
- Отрисовка кривых через `DrawCurve` (сглаженные линии)
- Управление анимацией через отдельный рабочий поток (`CWinThread`)

## Функциональные возможности (общие для обеих веток)

1. Построение графика - Отображение биений двух гармонических сигналов
2. Регулировка параметров - Поля ввода: частота (Omega) и разность частот (dOmega)
3. Анимация - Динамическое изменение фазы сигнала с визуализацией "движения"
